Event Management and Ticket Booking System
Project Overview
This is an ASP.NET Core MVC-based web application for managing events and booking tickets.
The system supports the following functionalities:

Event Creation and Management (Organizers).
Real-time seat availability and booking (Users).
User registration and login functionality.
Multiple ticket types (Regular, VIP).
Cart functionality for multiple ticket purchases.
Technologies Used
Framework: ASP.NET Core MVC
Language: C#
Database: SQLite / SQL Server (based on configuration)
Front-End: Razor Views, HTML5, CSS3, Bootstrap
Tools: Visual Studio 2022 / JetBrains Rider
ORM: Entity Framework Core
